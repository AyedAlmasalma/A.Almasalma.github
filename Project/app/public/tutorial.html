How to use the Decoder

In order to use the decoder all that is need a modern web browser and a DNG file. After the file has been uploaded the file is presented to the user. On the left there will be some general information about the file, in the middle there will be a picture from the file. Beneath the picture there are tow aditional buttons, one for downloading the JPEG, one for downloading the metadata. In order to start download the Metadata, the button "Download Metadata" should be clicked. Once the download is complete, the metadata for the DNG file will be displayed in the text editor.
